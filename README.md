# HR Analytics Dashboard – Power BI Project

This project is an interactive HR Analytics Dashboard developed using Power BI to analyze employee-related data and generate meaningful business insights. The dashboard helps visualize workforce distribution, salary trends, attendance performance, department-wise employee count, and country-based employee analysis through dynamic and interactive visualizations.

## Key Features

* Employee overview KPIs
* Salary distribution analysis
* Attendance performance tracking
* Department-wise employee insights
* Country-wise employee distribution using map visualization
* Interactive slicers and filters
* Clean and modern dashboard UI design

## Tools & Technologies Used

* Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Excel / CSV Dataset

## Dashboard Insights

* Average employee salary and workforce size
* Salary comparison by experience level and age
* Employee attendance analysis by gender and year
* Department-wise employee distribution
* Geographic distribution of employees across countries

## Project Goal

The main objective of this project is to demonstrate data cleaning, data modeling, DAX calculations, and dashboard designing skills by building a professional HR Analytics solution suitable for business reporting and decision-making.
